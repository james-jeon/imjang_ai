# AI 개발 시스템 비전

## 목적

기획, 테스트, 구현, 리뷰, 배포를 자동화하는 AI 기반 소프트웨어 개발 라이프사이클의 비전을 정의한다.

## 배경

서비스 전체 스택(앱, 웹, 백엔드) 리뉴얼을 진행하면서 AI를 개발 라이프사이클에 도입하는 아이디어가 제시되었다.

핵심 아이디어:
- 개발 작업은 Jira로 관리한다
- AI가 Jira 작업을 분석한다
- AI가 기획, 테스트, 코딩, 리뷰를 수행한다

## 핵심 개념

Jira가 개발 파이프라인의 시작점이 된다.

```
Jira → AI 분석 → 개발 자동화
```

## 목표

- 개발 생산성 향상
- 반복 작업 감소
- 문서 품질 개선
- 테스트 커버리지 향상

## 기대 효과

AI가 대부분의 구현 작업을 담당하고, 개발자는 검증과 아키텍처에 집중한다.

## 성과 측정

구체적인 품질 지표와 측정 방법은 12_quality_metrics를 참조한다.
목표 수치는 파일럿 운영 후 기준선(baseline)을 확보한 뒤 설정한다.
